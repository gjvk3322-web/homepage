# 🖼️ 시공 사례 갤러리 사진 업로드 가이드 (v2)

## 📁 폴더 구조 만들기

터미널에서 한 번에 4개 폴더 생성:

```bash
cd ~/homepage
mkdir -p img/cases/1000-modern
mkdir -p img/cases/500-modern
mkdir -p img/cases/500-marble
mkdir -p img/cases/500-cotton
```

---

## 📸 사진 파일명 규칙

## 📐 사진 권장 스펙

| 항목 | 권장 |
|---|---|
| **비율** | 16:10 (또는 4:3) |
| **해상도** | 1600 × 1000px 정도 (너무 크면 로딩 느림) |
| **용량** | 장당 200KB 이하로 압축 추천 |
| **포맷** | JPG (PNG는 너무 무거움) |
| **각도** | 거실 + 매트 한눈에 보이게 |

## 🎨 사진 선택 팁

좋은 시공 사례 = **다양한 인테리어 스타일** 보여주기

```
1000-modern (5장):
  - 미니멀 화이트 거실 (모델하우스 느낌)
  - 따뜻한 우드 톤 거실
  - 모던한 다크 인테리어 거실
  - 키즈룸 (애기 있는 가정)
  - 펫 가정 (강아지·고양이 보이는 사진)

500-modern (3장):
  - 좁은 거실 (소형 평수)
  - 넓은 공간 일부 시공
  - 부분 시공 (아일랜드 등)

500-marble (3장):
  - 고급스러운 마블 무늬가 잘 보이는 사진
  - 다양한 인테리어와 매칭

500-cotton (3장):
  - 따뜻한 베이지 톤이 잘 보이는 사진
  - 자연광 비치는 거실
```

## 💡 사진 압축 팁 (파일 크기 줄이기)

Mac에서 큰 사진은 **미리보기 앱**으로 압축:
```
1. 사진 미리보기로 열기
2. 파일 → 내보내기
3. 형식: JPEG
4. 품질: 60~70% (충분히 깔끔)
5. 크기 조절 (1600px 너비 정도)
```

또는 온라인:
- tinypng.com (무료, 드래그&드롭)
